# hallobitcoin.github.io

This repository uses Hugo with "Dot" as default theme.
[Hugo](https://gohugo.io) is a popular open-source static site generators.
[Check out Dot theme](https://docs.gethugothemes.com/dot) and see the [live demo](http://demo.themefisher.com/dot-hugo/).

## install
- install hugo

## build
```
[user@localhost hallobitcoin.github.io]$ hugo
```

## build with live reload
```
[user@localhost hallobitcoin.github.io]$ hugo serve
```
example output:
```
Watching for changes in /home/user/workspace/hallobitcoin.github.io/{archetypes,content,data,i18n,layouts,static,themes}
Watching for config changes in /home/user/workspace/hallobitcoin.github.io/config.toml
Environment: "development"
Serving pages from memory
Running in Fast Render Mode. For full rebuilds on change: hugo server --disableFastRender
Web Server is available at http://localhost:1313/ (bind address 127.0.0.1)
```

Visit [http://localhost:1313](http://localhost:1313) and it will auto-reload the page immediately if the content of a file changed.
